## 课程内容
**开发环境：**
> Windows 10，Python 3.6，Scrapy 1.5，MongoDB 3.6，PyCharm

**适用人群：**

> 1. 具有一定的 Python 基础
> 2. 具有一定的 Linux 系统管理基础（编译安装软件、Yum 包管理工具等）
> 3. 具有一定的 Mongodb 数据库管理基础（增删改查）



**课程大纲：**

- Scrapy 介绍
- Scrapy 的安装
- Scrapy 在安装时经常遇到的问题
- MongoDB 的安装
- Scrapy 框架、组件、数据流
- Scrapy 数据抓取实战



### 一、Scrpy 安装

